Detailed Description

template<class T>
class cMsgDispatcher< T >

This class provides a cMsg C callback method that dispatchs to a member function of another object, used internally.

This object extends cMsgCallback and thus has a callback method.
It further stores a pointer to another object and to a member function of the object. That member function is invoked by the callback method of this object.
